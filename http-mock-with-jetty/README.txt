==============================================
     Armando un mock de un servidor HTTP 
                 http://www.dosideas.com
==============================================

Este proyecto contiene un pequeño ejemplo sobre el uso de Jetty con JUnit para crear
un mock de un servidor HTTP.


 Contenido
===========
 src/                 : directorio con los fuentes del proyecto.
 test/                : pruebas JUnit que compruebas las validaciones.


 Ejecución
===========
 El proyecto puede abrirse directamente con cualquier IDE y ejecutar la prueba unitaria
para comprobar el funcionamiento.
